Sistema de Registro e Pagamentos Simples

Este projeto foi desenvolvido para o registro e controle de pagamentos empresariais simples, permitindo o gerenciamento de clientes, serviços prestados e valores pagos. Sistema de Consulta de Clima por CEP

Este projeto foi desenvolvido para permitir a consulta de informações climáticas a partir do CEP informado pelo usuário. A aplicação utiliza a API ViaCEP para obter os dados de localização e a API OpenWeather para exibir informações em tempo real, como temperatura, condição climática, umidade e velocidade do vento, oferecendo uma experiência simples, rápida e intuitiva.

  • JavaScript
  • HTML
  • CSS
  • API ViaCEP
  • API OpenWeather

Sistema de Controle de Pedágio

Este projeto foi desenvolvido para o registro e controle de veículos em praças de pedágio. O sistema permite cadastrar a placa do veículo, registrar horários de entrada e saída entre os pontos de cobrança e calcular automaticamente o tempo de percurso, a velocidade média e o valor a ser pago. Ao final do turno, a aplicação gera um relatório com informações consolidadas, como total arrecadado, menor, maior e média de velocidade, proporcionando mais organização, precisão e controle operacional.

  • JavaScript
  • HTML
  • CSS
  • Lógica de Programação
